Consignas práctibles dentro del Html
1. El sitio web debe poseer al menos 4 páginas HTML o 4 secciones si se trata de un sitio One Page En mi caso he elegido crear 4 HTML
2. El sitio web debe incluir un formulario de contacto, con al menos 5 campos que incluya (un checkbox o radiobutton), un select y una imagen, con validación realizada mediante JavaScript para que los campos sean obligatorios. En la pagina del formulario se incluyen 5 campos y tambien hay una validación para todos salvo para el comentario, el cual es opcional
3. Tener al menos una página totalmente responsive con mínimo 3 puntos de corte con media querys para 3 tamaños de dispositivos (PC escritorio, Tablet, Celular). Para el diseño debe usarse CSS. Todas las paginas del tp intentan ser responsivas ya que se ha modiciado directamente el body en el main del css, esta modificación se encuentra al inicio del archivo
4. Utilizar al menos una animación, transformación o una transición. Se han utilizado 2 animaciones, las 2 en diferentes titulos de las paginas html, el primero en "Formulario de inscripcion" del html "Formulario" y el segundo en "Mis redes" en el html Contacto. Ambos metodos fueron usando "animate style".
